# 🎮 Micro Social Playground

> 交互式练习平台 - 在浏览器中边学边练，搭建你的社交平台

## ✨ 特点

- 🖥️ 内置代码编辑器（CodeMirror）
- 👁️ 实时预览效果
- ✅ 自动答案检查
- 💾 进度自动保存（localStorage）
- 📦 一键导出成品（ZIP 下载）
- 📱 支持手机/桌面预览切换

## 🚀 使用方式

直接用浏览器打开：

```bash
# 方式一：直接打开
open playground/index.html

# 方式二：用 HTTP 服务器（推荐）
cd playground
npx serve .
# 或
python -m http.server 8080
```

## 🗺️ 关卡列表

| # | 关卡 | 技能 | 难度 |
|---|------|------|------|
| 1 | 搭建页面骨架 | HTML 基础 | ⭐ |
| 2 | 三栏布局 | CSS Flexbox | ⭐ |
| 3 | 推文卡片组件 | HTML + CSS | ⭐⭐ |
| 4 | 发布推文框 | 表单 + CSS | ⭐ |
| 5 | 导航栏美化 | CSS 进阶 | ⭐⭐ |
| 6 | JavaScript 动态渲染 | JavaScript | ⭐⭐ |
| 7 | 点赞功能 | JS 交互 | ⭐⭐ |
| 8 | 发布推文功能 | JS 交互进阶 | ⭐⭐ |
| 9 | 数据持久化 | Web Storage | ⭐⭐⭐ |
| 10 | 🎉 完整社交平台 | 综合实战 | ⭐⭐⭐ |

## 🎯 学习路径

```
HTML 骨架 → CSS 布局 → 组件设计 → 表单交互
    → JavaScript 渲染 → 事件处理 → 数据存储
        → 🎉 完整成品
```

## 📦 导出成品

完成所有关卡后，点击右上角 **📦 导出成品** 按钮：

- 下载包含 HTML/CSS/JS 的 ZIP 文件
- 直接用浏览器打开即可运行
- 可以部署到局域网给朋友使用
