# Python Track / Python 路线

欢迎来到 Python 技术路线！这里提供两条路径：

## 🟢 Flask 路线（轻量灵活）

适合想深入理解 Web 原理的开发者。

```
Level 1: Flask Basic ────────> 静态页面 + 模板
    │
    ▼
Level 2: Flask API ──────────> RESTful API
    │
    ▼
选择 Django 进阶或继续 Flask 扩展
```

**特点：**
- 简单易学
- 灵活自由
- 适合小项目

## 🔵 Django 路线（全功能框架）

适合想快速开发完整应用的开发者。

```
Level 3: Django Basic ───────> Django 基础
    │
    ▼
Level 4: Django Full ────────> REST API + WebSocket
```

**特点：**
- 内置功能丰富
- Admin 管理后台
- ORM 强大
- 适合大项目

## 📦 快速对比

| 特性 | Flask | Django |
|------|-------|--------|
| 复杂度 | 低 | 高 |
| 灵活性 | 高 | 中 |
| 内置功能 | 少 | 多 |
| 学习曲线 | 平缓 | 陡峭 |
| 适合场景 | API/微服务 | 完整应用 |

## 🚀 选择建议

- **新手推荐**: Django（功能齐全，文档丰富）
- **喜欢精简**: Flask（自己选组件）
- **只想做 API**: Flask + Flask-RESTful
- **完整社交平台**: Django + DRF
